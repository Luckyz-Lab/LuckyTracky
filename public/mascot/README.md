# 🐱 Mascot Lottie Files

วางไฟล์ Lottie animation (.json) ที่นี่ — ถ้าไม่มีไฟล์ แอปจะแสดง SVG fallback แทนโดยอัตโนมัติ

## Slot ที่รองรับ

| ชื่อไฟล์        | คำอธิบาย                              | ค้นหาใน LottieFiles            |
|----------------|--------------------------------------|-------------------------------|
| `idle.json`    | แมวนั่งเฉยๆ หายใจ / ตาขยิบ           | "cute cat idle loop"          |
| `loading.json` | แมววิ่งหรือเดินดุ๊กดิ๊ก               | "cat running loop"            |
| `sleeping.json`| แมวนอนหลับ มี zZz                    | "cat sleeping animation"      |
| `walking.json` | แมวเดิน                               | "cat walk cycle"              |
| `celebrate.json`| แมวกระโดด/ฉลอง/โบกมือ               | "cat celebrate happy"         |
| `eating.json`  | แมวกินข้าว / เลียปาก                  | "cat eating"                  |
| `tap.json`     | แมวสะดุ้ง / ตอบสนอง                  | "cat surprised reaction"      |
| `shocked.json` | แมวตกใจ ตาเบิกกว้าง                  | "cat shocked eyes"            |

## แนะนำแหล่ง

- [LottieFiles.com](https://lottiefiles.com) — ค้น "orange cat" หรือ "cute cat"
- [IconScout.com](https://iconscout.com) — ค้น "cat animation"

## วิธีใช้

1. ดาวน์โหลดไฟล์ `.json` จาก LottieFiles
2. วางในโฟลเดอร์นี้ ตั้งชื่อตาม slot เช่น `idle.json`
3. แอปจะโหลดอัตโนมัติ ไม่ต้องแก้โค้ด

> **Note:** ถ้าไม่มีไฟล์ แอปจะแสดง SVG fallback แมวส้มฮู้ดโคลเวอร์ที่วาดไว้แล้ว
