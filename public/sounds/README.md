# 🔊 Sound Effects

วางไฟล์เสียง `.mp3` ที่นี่ — ถ้าไม่มีไฟล์ แอปจะไม่มีเสียงแต่ยังทำงานได้ปกติ
เสียงปิดเป็นค่าเริ่มต้น (เปิดได้ที่ปุ่ม 🔊 ใน Sidebar)

## ไฟล์ที่รองรับ

| ชื่อไฟล์        | ใช้เมื่อ                                |
|----------------|----------------------------------------|
| `tap.mp3`      | กดปุ่ม / แตะมาสคอต                     |
| `coin.mp3`     | บันทึกรายการสำเร็จ                      |
| `celebrate.mp3`| บรรลุเป้าหมายออม / งบเต็ม               |
| `alert.mp3`    | เตือนงบเกิน / เงินใกล้หมด              |
| `swoosh.mp3`   | เปิด/ปิด panel                         |

## แนะนำแหล่งเสียงฟรี

- [Pixabay.com/sound-effects](https://pixabay.com/sound-effects/) — ค้น "coin", "pop", "success"
- [Freesound.org](https://freesound.org) — เสียงฟรี Creative Commons
- [Zapsplat.com](https://www.zapsplat.com) — ต้องสมัครฟรี

## ข้อควรระวัง

- ใช้ไฟล์สั้น (< 2 วินาที) เพื่อ UX ที่ดี
- volume ถูกตั้งที่ 35% โดยอัตโนมัติ
